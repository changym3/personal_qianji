# AssetChange · 资迹 · 资产负债表

一个轻量的个人资产负债表管理工具，支持网页端使用，数据保存在浏览器本地。

## 功能特性

### 页面导航

- **顶部 Header 固定** — 「资迹 · 资产负债表」标题、账本切换、导入/导出/重置按钮始终固定在页面顶部
- **Tab 栏固定** — 「趋势图表」「当前资产」Tab 标签及右侧全局设置按钮在页面滚动时固定在 Header 下方，随时可切换 Tab 或调整设置

### 全局设置（Tab 栏右侧）

- **金额单位** — 「元」与「万」切换，对「趋势图表」和「当前资产」两个 Tab 同时生效；图表、卡片、Tooltip 全部同步
- **家庭资产计入模式** — 三种模式下拉选择，全局生效：
  - **不计入** — 家庭资产条目完全排除在净资产统计之外
  - **计入负债** — 家庭资产总额视为一笔负债，增加总负债，降低净资产
  - **从资产扣除** — 家庭资产总额从总资产中扣除，净资产同样降低；两种「计入」方式净资产结果相同，区别在于总资产/总负债的分配

### 趋势图表（默认首页）

- **配置记忆** — 聚合方式、趋势指标、时间范围预设保存在本地，下次打开沿用（自定义日期范围仅在当前会话内有效）
- **面积趋势图** — 展示净资产/总资产/总负债的面积图，支持三种指标切换，颜色自动跟随
- **时间聚合** — 支持按快照（默认）、按周、按月；按周/按月时每个周期只保留该周期内**最后一次**快照
- **时间范围筛选** — 默认展示全部数据，支持快捷筛选（近一月、近三月、近半年、近一年）和自定义起止日期
- **时间滚动条** — 图表下方提供 Brush 滚动条，可拖拽选择查看范围
- **悬浮详情** — 鼠标悬浮时 Tooltip 同时展示净资产、总资产、总负债三项数据
- **快照卡片时间线** — 每个快照展示为卡片，显示完整日期、净资产、相比上次快照变化百分比/金额、资产负债进度条（蓝色=净资产占比，橙色=负债占比）、备注（若有）

### 当前资产（第二 Tab）

- **与最新快照一致** — 进入后展示「资产时间线」中最新一张快照对应的详情编辑页
- **无快照提示** — 若尚无任何快照，会提示先在趋势页创建

### 快照管理（资产卡片详情页）

- **四大类并列** — 详情页分为「总资产」「总负债」「家庭资产」「备注」四大块，各块可点击标题行整块收起/展开
- **家庭资产独立管理** — 家庭资产与总资产/总负债平级，支持相同的分组和条目操作；条目图标为 🏠；计入模式标签显示在大类标题旁
- **分组操作** — 三大资产类别均支持添加/删除/重命名分组，分组间支持拖拽排序
- **条目操作** — 支持添加/编辑/删除条目；条目支持在三大类之间跨类拖拽；Enter 快速确认
- **条目标记** — 每个条目可独立标记「不计入总资产」（黄色徽标）和「计入家庭资产」（蓝色徽标）；家庭资产大类下新增条目时「计入家庭资产」默认开启
- **实时汇总** — 顶部根据全局「家庭资产计入模式」动态计算净资产/总资产/总负债并实时展示
- **快照备注** — 底部备注区平时展示文本，点击进入编辑，失焦自动保存；备注内容同步显示在资产时间线卡片上
- **卡片日期直改** — 在趋势页快照卡片上可直接修改日期，自动校验是否与已有快照冲突
- **删除快照** — 资产时间线每张卡片支持直接删除（带确认弹窗）
- **新建安全保护** — 新建快照的默认日期自动避开已有快照；保存时若日期冲突则提示修改，不会静默覆盖已有数据
- **空白快照** — 三大类别各默认包含一个「未分组」分组及「未命名」条目

### 多账本

- **独立数据** — 每个账本拥有独立的资产负债与快照数据
- **顶栏切换** — 通过账本下拉切换；支持新增账本、重命名、删除（至少保留一个账本）
- **新建即进入** — 新建账本后自动切换到该账本

### 数据管理

- **本地持久化** — 数据自动保存在浏览器 LocalStorage 中，无需后端
- **按账本存储** — 导入/导出、重置均针对当前选中账本
- **导入/导出** — 支持 JSON 格式的数据备份与恢复
- **重置** — 一键清空当前账本数据

## 技术栈

- React 19 + TypeScript
- Vite 8（构建工具）
- Ant Design 6（UI 组件库）
- Recharts 3（图表）
- Zustand 5（状态管理 + LocalStorage 持久化）
- @dnd-kit（拖拽排序）
- Day.js（日期处理）

## 快速开始

### 环境要求

- Node.js >= 18
- npm >= 9

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

启动后在浏览器访问 http://localhost:5173

### 构建生产版本

```bash
npm run build
```

构建产物输出到 `dist/` 目录。

### 预览生产版本

```bash
npm run preview
```

## 项目结构

```
src/
  types/index.ts                    # TypeScript 类型定义（Account, Category, Group, Item, WeeklySnapshot）
  store/useAccountStore.ts          # Zustand：多账本 + 当前账本数据 + LocalStorage 持久化
  store/useChartSettingsStore.ts    # 全局设置（金额单位、家庭资产模式、聚合/指标/时间预设）持久化
  utils/snapshot.ts                 # 快照工具（日期归一、汇总、upsert）
  components/
    BalanceSheet/                   # 资产负债表相关组件（可供扩展或复用）
    Chart/
      TrendChart.tsx                # 趋势图表（面积图 + 卡片时间线 + 配置区）
      SnapshotEditor.tsx            # 快照编辑器（新增/编辑快照，含家庭资产大类）
    CurrentAsset/
      index.tsx                     # 「当前资产」Tab：最新快照详情
    LedgerDropdown/
      index.tsx                     # 顶栏账本切换与管理
  App.tsx                           # 主入口（Tab + 顶栏 + 全局设置控件）
  main.tsx                          # 应用入口
  index.css                         # 全局样式 + 响应式适配
```

## 使用说明

1. **切换账本** — 在顶栏账本下拉中选择、新建或重命名账本；数据按账本隔离
2. **设置金额单位** — 在 Tab 栏右侧点击「金额单位」下拉，选择「元」或「万」，全局生效
3. **设置家庭资产模式** — 在 Tab 栏右侧点击「家庭资产」下拉，选择计入方式，全局生效
4. **查看趋势** — 默认进入趋势图表，在「配置」中设置聚合方式和时间范围，在「资产趋势」查看变化
5. **新增快照** — 点击「新增快照」卡片进入编辑器，分别填写总资产、总负债、家庭资产的分组和条目后保存
6. **编辑快照** — 在资产时间线点击卡片进入「资产卡片详情页」，或在「当前资产」Tab 直接编辑最新快照
7. **导入/导出** — 使用顶栏导出备份当前账本，导入覆盖当前账本数据
